# Student Registry

A modern, responsive web application for managing student records efficiently. Built with React, TypeScript, and Tailwind CSS, this application provides a seamless experience for administrators to handle student data.

## Features

-   **Student Management**: Add, edit, and delete student records with ease.
-   **Detailed Profiles**: View comprehensive details for each student, including personal info, parent details, and guardian contact information.
-   **Search & Filter**: Quickly find students by name or enrollment number.
-   **Data Export**: Export student data to CSV format for external use or reporting.
-   **Multi-Language Support**: Toggle between English and Burmese (Myanmar) languages.
-   **Dark Mode**: Built-in dark mode for comfortable viewing in low-light environments.
-   **Responsive Design**: Optimized for both desktop (table view) and mobile devices (card view).
-   **Interactive UI**: Smooth animations and transitions powered by Framer Motion.
-   **Form Validation**: Multi-step form with validation to ensure data integrity.

## Tech Stack

-   **Frontend**: React (Vite), TypeScript
-   **Styling**: Tailwind CSS
-   **Icons**: Lucide React
-   **Animations**: Framer Motion
-   **Backend/Database**: Supabase (PostgreSQL)

## Getting Started

### Prerequisites

-   Node.js (v18 or higher)
-   npm or yarn
-   A Supabase account and project

### Installation

1.  **Clone the repository:**

    ```bash
    git clone <repository-url>
    cd student-registry
    ```

2.  **Install dependencies:**

    ```bash
    npm install
    ```

3.  **Environment Configuration:**

    Create a `.env` file in the root directory and add your Supabase credentials:

    ```env
    VITE_SUPABASE_URL=your_supabase_project_url
    VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
    ```

4.  **Database Setup:**

    Run the following SQL in your Supabase SQL Editor to create the `students` table:

    ```sql
    create table students (
      id bigint generated by default as identity primary key,
      created_at timestamp with time zone default timezone('utc'::text, now()) not null,
      name text not null,
      enrollment_no text not null,
      grade text,
      gender text,
      birth_date date,
      father_name text,
      mother_name text,
      phone_no text,
      address text,
      guardian_name text,
      guardian_phone text,
      guardian_address text,
      notes text
    );
    ```

5.  **Run the development server:**

    ```bash
    npm run dev
    ```

    Open your browser and navigate to `http://localhost:5173` (or the port shown in your terminal).

## Usage

-   **Dashboard**: The main dashboard displays a list of all students. Use the search bar to filter the list.
-   **Add Student**: Click the "Add Student" button to open the multi-step registration form.
-   **Edit/Delete**: Use the action buttons on each student row (desktop) or card (mobile) to edit or delete a record.
-   **Export**: Click the "Export CSV" button to download the current list of students.
-   **Settings**: Use the toggle buttons in the header to switch between Dark/Light mode and English/Burmese language.
